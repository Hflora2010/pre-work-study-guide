HTML

  • HTML stands for Hypertext Markup Language.
  • Is one of the three cornerstone languages used in web development.
  • Made up of elements which work together to provide content and makeup the skeleton/framework of a webpage.
  • Elements are case sensitive.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An HTML attribute defines additional information about the element.
  • HTML attributes are always defined at the beginning of an element, never the closing.

Git

  • A version control system that lets you manage and keep track of each version of your project.
  • Git commands always start with git.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [url] retrives an entire repository from a hosted location via URL.
  • git add -A stages the changes, indicating to git that we want to save the changes to that file.
